# Python Programming - Резюме

## Основная информация

Python — это высокоуровневый язык программирования общего назначения, созданный Гвидо ван Россумом в конце 1980-х годов. Язык характеризуется философией дизайна, которая подчеркивает читаемость кода через использование значимых отступов.

## Ключевые факты

• **Типизация и парадигмы**: Python поддерживает динамическую типизацию, автоматическую сборку мусора и множественные парадигмы программирования (структурное, объектно-ориентированное и функциональное программирование)

• **Актуальные версии**: По состоянию на 2025 год Python Software Foundation поддерживает версии 3.10, 3.11, 3.12, 3.13 и 3.14, следуя политике ежегодных релизов и пятилетней поддержки

• **Эволюция языка**: Python 3.0 (2008) стал крупной ревизией без полной обратной совместимости. Начиная с версии 3.5 добавлена поддержка опциональной статической типизации

• **Популярность в ML**: Python получил широкое распространение в сообществе машинного обучения благодаря мощным библиотекам

• **Богатая экосистема**: Включает специализированные инструменты как Pydoc (документирование), NumPy (работа с массивами и матрицами) и SciPy (научные вычисления)
