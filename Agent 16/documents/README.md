# AI Агент для анализа данных

## Описание проекта

Это проект по созданию AI агента, который помогает анализировать большие объемы текстовых данных. Агент использует современные технологии машинного обучения для извлечения полезной информации из документов.

## Основные возможности

### 1. Обработка документов
Система может обрабатывать различные форматы документов:
- Текстовые файлы (.txt)
- Markdown документы (.md)
- PDF файлы
- Веб-страницы (HTML)

### 2. Семантический поиск
Агент использует векторные эмбеддинги для понимания смысла текста. Это позволяет находить релевантную информацию даже если точные слова не совпадают.

### 3. Генерация ответов
С помощью больших языковых моделей агент может генерировать ответы на вопросы, основываясь на проиндексированных документах.

## Технологический стек

- **Python 3.11** - основной язык программирования
- **Sentence Transformers** - для создания эмбеддингов
- **FAISS** - для эффективного векторного поиска
- **LangChain** - для работы с языковыми моделями
- **Ollama** - локальный запуск LLM моделей

## Архитектура системы

Система состоит из трех основных компонентов:

1. **Индексатор** - разбивает документы на чанки и создает эмбеддинги
2. **Поисковик** - находит релевантные фрагменты по запросу
3. **Генератор** - формирует финальный ответ на основе найденной информации

## Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/username/ai-agent.git

# Установите зависимости
pip install -r requirements.txt

# Запустите индексацию
python index_documents.py

# Запустите агента
python main.py
```

## Примеры использования

### Индексация документов
```python
from indexer import DocumentIndexer

indexer = DocumentIndexer()
indexer.index_folder("./documents")
indexer.save_index("my_index.json")
```

### Поиск информации
```python
from search import SemanticSearch

search = SemanticSearch("my_index.json")
results = search.query("Как работает RAG?", top_k=5)
```

### Генерация ответа
```python
from agent import AIAgent

agent = AIAgent(index_path="my_index.json")
answer = agent.ask("Объясни концепцию эмбеддингов")
print(answer)
```

## Производительность

На тестовом датасете из 1000 документов:
- Индексация: ~5 минут
- Поиск: ~0.1 секунды
- Генерация ответа: ~2-3 секунды
